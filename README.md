# Customer Segmentation using Machine Learning

## 📌 Description
This project uses Machine Learning (K-Means Clustering) to segment customers based on their purchasing behavior.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Elbow Method for optimal clusters
- Customer segmentation using K-Means
- Visualization of clusters

## 📂 Files Included
- customer_segmentation.ipynb
- dataset (.csv files)
- visualizations (.png)
- project_report.docx

## ▶️ How to Run
1. Install required libraries:
   pip install pandas numpy matplotlib scikit-learn

2. Open Jupyter Notebook:
   jupyter notebook

3. Run the notebook file

## 🎯 Output
- Clustered customers
- Visual graphs showing segmentation
